# Gituro Java Development 1 Project - CharacterBattler
Final project untuk course 1 batch one dari Gituro.
CharacterBattler adalah sebuah aplikasi game text-based yang berjalan pada CLI, dibuat menggunakan Java.  
Diharapkan project ini akan dapat mengajarkan konsep-konsep paling penting dari pengembangan Java dan OOP.

# Features
- Register multiple characters
- Leveling, stats, moves system
- Turn-based battle
